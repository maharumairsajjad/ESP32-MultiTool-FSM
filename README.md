# ESP32 Multi-Tool (FSM Based)

This project is an ESP32-based multi-functional embedded system designed using a Finite State Machine (FSM).

## Features
- Distance Measurement (Ultrasonic Sensor)
- Servo Control
- Thermostat with Alarm (DHT11)
- Timer Alarm (15s, 30s, 45s, 60s)
- OLED Display (SSD1306)
- Rotary Encoder Navigation
- Buzzer Alerts

## Hardware Used
- ESP32
- OLED SSD1306 (128x64)
- Rotary Encoder
- Ultrasonic Sensor (HC-SR04)
- Servo Motor
- DHT11
- Buzzer
